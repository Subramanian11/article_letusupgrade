ARTICLE

Web address:

Web address is the address by wich the website of a firm or organization or an institution or an individual is identified on the internet,. i.e the domain name of the website, which is unique.

Example:

website1
website2

Website:

Web site is a collection of web pages which consists of information about an individual, institution or organization and so on.

There are two types of web addressing methods.

1.Numbering Method:

This consists of a set of 4 numbers each separated by a dot.

2.Naming Method:

This type of addressing methods is easy and simple to use.

Example:

website1
website2
